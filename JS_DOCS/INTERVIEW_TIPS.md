# 💡 面接での勝ちパターン (JavaScript編)

これらのファイルについて聞かれたら、こう答えてください。

> 「JavaScriptは、単に動きをつけるだけでなく、**ユーザーの体験(UX)** を損なわないように設計しました。
>
> 1.  **`base.js`**: スマホでもPCでも操作しやすいUIと、PWAによるアプリ化。
> 2.  **`calendar.js`**: **Fetch API**を使って画面遷移なしでデータを保存し、アプリのようなサクサク感を実現。
> 3.  **誤操作防止**: そもそも権限がない人にはボタンを表示しない(編集させない)設計。
> 4.  **`sw.js`**: **Service Worker**でオフライン時の挙動まで考慮。
>
> ただ動けばいいだけでなく、**『使う人がストレスを感じないか』** を常に考えて実装しました。」
