## ⚔️ 2. あなたの「最強の武器」3選
これだけは自信満々に語ってください。

### 武器①：PWA (Progressive Web App)
*   **アピール**: 「Webサイトなのに、スマホアプリとしてインストールできます」
*   **効果**: アイコンがホーム画面に追加され、アドレスバーも消える。起動も速い。

### 武器②：Optimistic UI (楽観的UI)
*   **アピール**: 「参加ボタンを押した瞬間、通信完了を待たずに色が変わり、サクサク動きます」
*   **効果**: ネイティブアプリ負けない操作感。実は裏でJavaScriptが頑張って通信している。

### 武器③：N+1問題の解消 (パフォーマンス)
*   **アピール**: 「データが増えても重くならないようにチューニングしました」
*   **具体例**: 「最初は参加者全員のデータを取るのに100回SQLが走っていましたが、Djangoの `select_related` を使って**1発のSQL**で取れるようにしました」
